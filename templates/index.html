API使用:
  • 默认使用中文模板 (language="zh")
  • 可在请求中指定 language 参数选择模板
  • 支持的语言: zh (中文), en (英文)
  • 每个语言只能有一个激活模板
示例请求: 
{
  "text": "输入文本",
  "schema": {
    "schema": "medical",
    "triplet": ["疾病-症状->症状"]
  },
  "language": "zh"  // 可选，默认为 "zh"
}
